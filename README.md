# Bukidnon Explorer

## Overview

Bukidnon Explorer is a responsive web application that showcases tourist destinations and cultural attractions in Bukidnon, Philippines. This platform provides comprehensive information about local landmarks, natural wonders, and cultural events to enhance visitor experience and promote regional tourism.

## Key Features

### Core Functionality
- **Homepage**: Central hub displaying featured destinations and navigation
- **Destination Pages**: Dedicated pages for major attractions including Lake Apo, Dahilayan Adventure Park, Kaamulan Festival, Kipolot Falls, Cedar Falls, Sabangan Falls, and Nasuli Communal Forest
- **Blog Section**: Informational content about local destinations
- **About Page**: Project information and background
- **Contact Form**: User inquiry submission system
- **Authentication System**: User login and registration interfaces

### Technical Features
- Fully responsive design compatible with mobile, tablet, and desktop devices
- Nature-inspired color scheme optimized for tourism content
- Poppins font family for enhanced readability
- Smooth CSS animations and transitions
- Font Awesome icon integration

## Installation and Setup

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Optional: Local development server for optimal performance

### Installation Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/JohnCharlieCabilin-27/Buklocalspot.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Buklocalspot
   ```

3. Launch the application:
   - Option A: Open `index.html` directly in your web browser
   - Option B: Use a local development server (recommended)

### Development Server Setup

To run the application with Live Server in Visual Studio Code:

1. Install the Live Server extension from the VS Code marketplace
2. Open the project folder in VS Code
3. Right-click `index.html` and select "Open with Live Server"
4. The application will launch automatically at `http://localhost:5500`
